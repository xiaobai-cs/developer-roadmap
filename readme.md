## 开发

克隆存储库，安装依赖项并启动应用程序

```bash
git clone git@github.com:kamranahmedse/developer-roadmap.git
npm install
npm run dev
```

## 贡献

> 查看 [贡献文档](./contributing.md) ，了解如何更新任何路线图。

- 对现有路线图提出更改建议
- 在 Issue 中讨论想法
